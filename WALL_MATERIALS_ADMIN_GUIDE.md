## FORM FIELDS MAPPING

```
1. Root Category    →  "Wall"
2. Classification   →  "Load Bearing" OR "Non-Load Bearing"
3. Material Type    →  "Brick", "Block", "Stone", "Cement", or "Sand"
4. Product Name     →  Brand + Material type (e.g., "Wienerberger First Class Clay Brick")
5. Grade/Spec       →  Grade info (e.g., "First Class", "2500 PSI", "AAC 600", "PPC")
6. Dimensions       →  For materials only: L × W × H in inches (e.g., "9x4.25x3")
7. Image URL        →  Product image link
8. Price per Unit   →  ₹ amount
9. Unit             →  From dropdown: "Nos", "Bag (50kg)", "Cubic Feet (cft)", etc.
```

## Instructions for Admin Panel

1. **Go to Admin Dashboard**
   - URL: http://localhost:3000/admin (or your admin URL)

2. **Click "Add New Material"**

3. **Fill Form (Example - Wienerberger):**
   - Root Category: Select "Wall"
   - Classification: Select "Load Bearing"
   - Material Type: Select "Brick"
   - Product Display Name: "Wienerberger First Class Clay Brick"
   - Specification/Grade: "First Class"
   - Block Dimensions: "9x4.25x3"
   - Image URL: [copy-paste from above]
   - Market Price: 8.50
   - Unit: Select "Nos (Numbers)"

4. **Click Save**

5. **Repeat for all 19 materials**

---

## Testing After Adding Materials

1. Go to User App → WallScreen
2. Tap "Select Material" under Load-Bearing Walls
3. Should see all your new materials with:
   - ✅ Images
   - ✅ Names
   - ✅ Prices per unit
   - ✅ Dimensions
   - ✅ Ratings & Reviews
